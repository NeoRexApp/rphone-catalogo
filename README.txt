RPHONE Web Starter (UI)
======================

Estructura recomendada:
- index.html, catalog.html, tracker.html (en la raíz)
- css/styles.css
- js/*.js
- assets/rphone-logo.svg

Cómo correr:
1) VS Code -> instala "Live Server"
2) Abre index.html con Live Server (NO uses file://)

Cambios rápidos:
- js/settings.js: cambia whatsappE164, address, hours.

Siguiente paso (cuando tú digas):
- Conectar Firebase Firestore + Storage para que catálogo/ofertas/faq/tickets salgan de la BD.
